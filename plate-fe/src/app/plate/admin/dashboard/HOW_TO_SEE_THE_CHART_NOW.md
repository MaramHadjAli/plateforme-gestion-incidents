## 🎮 COMMENT INTERAGIR

### Survolez (Hover):
```
1. Passez votre souris sur un segment
2. Vous verrez un effet glow (lumineux)
3. Le segment se met en avant
4. Animation fluide
```

### Cliquez (Click):
```
1. Cliquez sur n'importe quel segment
2. Le graphique zoom (1.8x plus grand)
3. Change de Pie en Donut
4. Détails affichés au centre
5. Particules émises
6. Son joue (optionnel)
```

### Mode Toggle:
```
1. Bouton "Switch to Donut" (ou Pie)
2. Transforme le graphique
3. Animation fluide (600ms)
```

### Son:
```
1. Bouton 🔊 ou 🔇
2. Active/Désactive le son du clic
3. Sound joue à chaque interaction
```

## ❓ QUE FAIRE SI...

### "Je ne vois toujours rien"
```
1. Vérifier que F5 a bien chargé (attendre 3 sec)
2. Scrollez VRAIMENT vers le bas
3. Chercher "Analyse Cinématique"
4. Si toujours rien: Vérifier F12 (console) pour erreurs
```

### "J'ai une erreur dans la console"
```
1. Appuyer F12
2. Aller dans Console
3. Chercher les erreurs rouges
4. Contacter si problème persiste
```

### "Le graphique est vide"
```
1. Attendre plus longtemps le chargement
2. Vérifier que le dashboard a des données
3. Si pas de données, créer quelques incidents d'abord
```

### "Les animations sont saccadées"
```
1. C'est normal? Non! Vérifier:
2. F12 → Performance
3. Enregistrer 5 sec
4. Vérifier FPS (doit être 60)
```
